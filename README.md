# FreedomPay Windows FCC — Auth & Capture Integration Guide

WeKnow Group certification documentation for the FreedomPay **Freeway Commerce
Connect (FCC)** Windows integration, deployed in standalone mode. Covers system
overview, data-flow diagrams, setup/configuration, and a Level 1 troubleshooting
guide. Prepared to the FreedomPay Certification Documentation Requirements.

## Build

This project is authored for **Overleaf** (or any full TeX Live install).

- **Main document:** `WeKnow_Windows_FCC_Integration_Guide.tex`
- **Compiler:** pdfLaTeX
- **Format:** A4

On Overleaf, set the main document above and the compiler to *pdfLaTeX* in
*Menu → Settings*. All packages used (`inter`, `tcolorbox`, `tikz`, `listings`,
`tabularx`, `lastpage`, …) are part of the standard Overleaf TeX Live
distribution — no manual installation is required.
